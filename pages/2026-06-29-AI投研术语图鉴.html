01 · 估值财务
估值与财务指标
判断"涨的是EPS还是PE"的核心工具集
术语缩写全称释义与报告用法
市盈率（滚动） PE(TTM) Price-to-Earnings (Trailing Twelve Months) 股价 ÷ 过去12个月每股收益（EPS）。衡量"按当前盈利水平，多少年回本"。TTM=滚动最近四个季度，比静态PE更及时。
报告用法：寒武纪PE(TTM) 348倍，意味着按当前利润需348年回本——典型的高预期透支。
远期市盈率 PE_FWD Forward Price-to-Earnings 股价 ÷ 未来12个月预期EPS。用分析师一致预期来算，反映市场对未来盈利的定价。PE_FWD > PE(TTM) 说明市场预期盈利会下滑。
报告用法：比亚迪PE_FWD 44.37 > PE_TTM 26.31，说明市场预期近期盈利继续恶化——"线在弯"的信号。
市盈率相对盈利增长比率 PEG Price/Earnings-to-Growth PE ÷ 盈利增速（%）。PEG=1为合理，<1偏便宜，>1.5偏贵，>2严重高估。弥补了PE不看增速的缺陷——高PE配高增速可能不贵。
报告用法：中际旭创PEG 0.54（便宜）vs 绿的谐波PEG 9.22（泡沫），同样的高PE，增速差异决定了贵不贵。
市盈率通道 PE Bands PE Band / PE Channel 把历史PE按分位画成带状通道（如5%/25%/50%/75%/95%），叠加当前PE位置。判断当前估值在历史区间的高低位置。PE分位低≠便宜，要看EPS斜率。
报告用法：拓普集团PE Bands在31%低位，但Q1净利-2.5%——"便宜+EPS下行=线在弯，不是买点"。
每股收益 EPS Earnings Per Share 净利润 ÷ 总股本。每股赚多少钱。股价 = PE × EPS，搞清楚涨的是PE还是EPS是估值分析的核心。EPS斜率向上=盈利加速，向下=盈利恶化。
报告用法：北方华创2年涨246%，同期EPS仅涨60%——"涨的是PE不是EPS"是估值透支的铁证。
市净率 / 市销率 PB / PS Price-to-Book / Price-to-Sales PB = 股价 ÷ 每股净资产（适合重资产）；PS = 市值 ÷ 营收（适合亏损但高增长公司）。当PE失真（如扭亏）时用PB/PS辅助估值。
报告用法：京东PB 1.07/股息4.08%作为防御性配置；绿的谐波PS 117倍佐证"纯主题博弈"。
PE剪刀差 — PE Scissors Gap 两个板块PE的比值差距。当差距达到极端水平（如9倍），估值引力会驱动资金从高PE流向低PE，形成回归。是风格切换的前兆指标。
报告用法：AI上游A股PE中位数~125倍 vs 港股科技~14倍，差近9倍——"9倍剪刀差2-3年内必然回归"。
戴维斯双杀 — Davis Double Play 盈利下滑（EPS降）+ 估值收缩（PE降）同时发生，股价暴跌。反之"戴维斯双击"=盈利上升+估值扩张。高PE+高增速股票最危险：增速一旦不及预期，EPS和PE同时杀。
报告用法："PEG>1.5的高PE票，增速一旦不及预期就是戴维斯双杀"——估值分析师的核心风险提示。
复合年增长率 CAGR Compound Annual Growth Rate 一项指标在多年间的年均复合增速。比简单平均更准确反映增长趋势。报告用于衡量产业市场规模和增速预期。
报告用法：全球具身智能市场未来5年CAGR约73%，2030年达2388亿元——量产元年确认的核心依据。
同比 / 投资回报率 YoY / ROI Year-over-Year / Return on Investment YoY = 与去年同期比较；ROI = 投资回报率 = (收益-成本)/成本。报告用YoY衡量季度盈利增速，用ROI衡量云厂商AI投入的财务回报。
报告用法：四大云厂商AI capex ROI全为负（Meta -28%），95%企业未获实质回报——"橡皮筋"断点的依据。

02 · 技术分析
技术分析指标
盘面派"破位就跑"的纪律依据
术语缩写全称释义与报告用法
移动平均线 MA5/10/20 Moving Average (5/10/20-day) 过去N个交易日收盘价的平均值连成的线。MA5=5日线（短期），MA10=10日线（中短期），MA20=20日线（中期）。股价在均线之上=强势，跌破=走弱信号。
报告用法："14只龙头破MA5+MA10"是机器人板块退潮的核心盘面信号；寒武纪5日线1457是"命门"。
指数平滑异同移动平均线 MACD Moving Average Convergence Divergence 由快线(DIF)、慢线(DEA)、柱状线组成。金叉=DIF上穿DEA（买入信号）；死叉=DIF下穿DEA（卖出信号）；双死叉=日线+周线同时死叉（强烈看空）。
报告用法：中际旭创/新易盛MACD双死叉+主力巨量出逃=光模块主升浪结束信号。
量价背离 — Volume-Price Divergence 股价创新高但成交量萎缩，或股价涨但主力资金净流出。说明上涨缺乏资金支撑，是趋势减弱的预警。高位龙头放量跌+资金流出=最危险的背离。
报告用法：兆易创新10日主力净流出71亿但股价仍在涨——量价背离是"假繁荣"的检查项之一。
换手率 — Turnover Rate 当日成交量 ÷ 流通股本。反映交易活跃度。换手率高（>5%）说明资金交投活跃，可能是主力进出；低（<1%）说明关注度低。
报告用法：AI算力芯片换手率6.25% vs 人形机器人3.72%——算力资金共识更强的盘面佐证。
52周位置 — 52-Week Position 当前股价在过去52周（一年）最高价与最低价之间的位置百分比。接近100%=接近年内最高，接近0%=接近年内最低。结合PE分位判断高低位。
公式：52周位置 = (当前价 − 52周最低价) ÷ (52周最高价 − 52周最低价) × 100%
报告用法：算力4只标的52周位置87.5%~96.4%（全在高位），机器人23.1%~82.4%（严重分化）。
乖离率 BIAS Bias / Deviation Rate 股价偏离均线的百分比。(股价-MA)/MA×100%。乖离过大（如>7%）说明短期超买，有回归均线的压力。
报告用法：海光信息乖离7.9%——股价远在MA5之上，短期有洗盘回调压力。

图3 · MACD 金叉与死叉：短线冲浪手的买卖信号

DIF（快线）上穿DEA（慢线）=金叉看多；下穿=死叉看空。报告中的"双死叉"=日线+周线同时死叉，强烈看空

03 · 资金市场
资金面与市场结构
信号派"四层信号对齐"的资金维度
术语缩写全称释义与报告用法
主力资金 — Main Force / Institutional Flow 大单（通常单笔成交额>50万）的净买卖额。主力净流入=大单买入额-卖出额>0，说明机构/游资在进场；净流出=在撤退。5日/20日主力净额是判断资金趋势的关键。
报告用法：三花智控20日主力-53亿、兆威-62亿——"全线净流出"是机器人资金信号未亮的核心依据。
北向资金 / 南向资金 — Northbound / Southbound Capital 北向=港股资金通过沪深股通买A股（常被视为外资风向标）；南向=内地资金通过港股通买港股。北向流入A股=外资看好，南向流入港股=内资抄底。
报告用法：北向6月370亿+60%流入算力 vs 南向买半导体不买互联网——资金在硬科技间切换。
虹吸效应 — Siphon Effect 某一热门板块大量吸金，导致其他板块失血下跌。表现为成交占比极端集中、全市场仅少数个股上涨。当虹吸到极致（成交占比破历史天花板），往往是风格切换前兆。
报告用法：AI算力成交占比24%破历史天花板15%——虹吸到极致开始反噬消费端（苹果涨价）。
K型分裂 — K-Shaped Divergence 市场分化为两条相反的线：少数板块向上（K的上臂），多数板块向下（K的下臂）。表现为指数没跌但大部分个股在跌，"赚了指数不赚钱"。
报告用法：3.59万亿天量下仅22%个股上涨、85.8%个股下跌——典型K型分裂+假繁荣特征。
风格切换 — Style Rotation 市场主导风格从一类板块转向另一类（如从成长切价值、从高位切低位、从大盘切小盘）。通常由估值极端分化、资金拥挤到极致、宏观环境变化触发。
报告用法：5个风格切换确认信号一个未出，但"种子已埋下"——等成交占比回落到15%以下。
假繁荣 — False Prosperity 指数或龙头在涨，但大部分个股在跌、赚钱效应极差。四条特征同时出现=典型假繁荣：①主线涨但全市场跌 ②成交额大但集中 ③龙头市值超常识 ④高位龙头资金流出。
报告用法：73%个股在3000点、电子占成交42%、寒武纪PE 337x、兆易10日流出71亿——四条全中。

图4 · 虹吸效应与K型分裂：资金极端集中的市场结构

AI算力成交占比24%破历史天花板15%，资金像黑洞一样吸干其他板块——这是风格切换的前兆

04 · 半导体存储
半导体与存储产业链
AI上游定价权的物理基础
术语缩写全称释义与报告用法
高带宽内存 HBM High Bandwidth Memory 通过3D堆叠技术将多层DRAM芯片垂直封装，配合AI GPU（如NVIDIA H100）提供超高带宽。HBM晶圆占用面积是普通DRAM的2.5-4倍——产HBM越多，通用DRAM越少，上游自己制造了下游的短缺。
报告用法：HBM订单排到2027年、部分锁定到2028年——"定价权落上游是供需物理规律"。
动态随机存取存储器 DRAM Dynamic Random Access Memory 需要不断刷新保持数据的内存芯片，用于PC/手机/服务器的运行内存。AI服务器DRAM用量是传统的8-10倍，已贡献超50%的DRAM总需求。合约价是存储周期的核心指标。
报告用法：DRAM Q1合约价环比涨90-95%，缺口4.9%（15年最大）——存储超级周期的数据。
闪存 NAND NAND Flash Memory 断电后仍能保存数据的非易失性存储，用于SSD/手机存储。与DRAM并称存储两大主线。NAND合约价也是存储周期跟踪指标。
报告用法：NAND涨55-60%、缺口4.2%——与DRAM同步进入超级周期。
光模块 — Optical Module 实现光信号与电信号转换的器件，是数据中心互联的核心部件。AI训练需要大量高速光模块（400G/800G）。光模块退潮（龙头MACD双死叉+主力流出）是AI主升浪后期的信号。
报告用法：中际旭创/新易盛MACD双死叉+巨量出逃=光模块退潮=主线从"鸡犬升天"进入"选龙头"。
半导体设备 — Semiconductor Equipment 制造芯片的机器（刻蚀、薄膜沉积、离子注入等）。半导体设备厂是芯片产能扩张的"卖铲人"。国产替代是核心逻辑——北方华创、中微公司是国产半导体设备双龙头。
报告用法：北方华创PE 106倍+超机构目标价34%——"涨的是PE不是EPS"的典型。
存储模组 / 存储芯片 — Memory Module / Memory Chip 存储芯片=原厂生产的裸片（三星/海力士/美光/兆易）；存储模组=把芯片封装成可用的内存条/SSD（佰维/江波龙）。模组厂利润弹性大——涨价周期里毛利率暴增。
报告用法：佰维存储YTD+337%、兆易+261%——存储模组/芯片是AI上游最暴利的环节。
晶圆 — Wafer 制造集成电路的硅片基板。先进晶圆产能有限，三大原厂将70-80%转向HBM，导致通用DRAM产出减少。"晶圆占用面积"是衡量产能分配的关键指标。
报告用法：HBM晶圆占用面积是普通DRAM的2.5-4倍——"上游自己制造了下游的短缺"。
合约价 — Contract Price 存储芯片厂与客户签订的长期供货合同价格，区别于现货价。合约价环比变化是判断存储周期的核心指标——转跌=周期见顶，继续涨=景气延续。
报告用法：DRAM合约价环比转跌是存储超级周期见顶的确认信号之一。

05 · 物理AI
物理AI与机器人产业链
"方向对"的产业共识与"时机到"的分歧
术语缩写全称释义与报告用法
物理AI — Physical AI AI从数字世界进入物理世界的统称，包括人形机器人、自动驾驶、工业机器人智能化、无人机/低空经济。核心是AI具备了感知+决策+执行物理动作的能力。2026被定义为"量产元年"。
报告用法：物理AI是未来3-5年最确定产业方向，但4位专家"方向无分歧、时机0共识"。
具身智能 — Embodied Intelligence / Embodied AI AI通过物理身体与环境交互来学习和进化，区别于只有"大脑"的大语言模型。人形机器人是具身智能的典型载体。全球市场未来5年CAGR约73%。
报告用法：具身智能景气度★★★★★，生命周期处于"1到10启动期"。
谐波减速器 — Harmonic Reducer 人形机器人关节核心部件，利用谐波传动原理实现高精度、大减速比。绿的谐波国内市占>60%。是机器人产业链"卡位最确定"的环节之一，但当下PE极高（519倍）。
报告用法：谐波减速器卡位★★★★★，但绿的谐波PE 519/PEG 6.45——"卡位好但太贵，等跌下来"。
RV减速器 — Rotary Vector Reducer 另一种精密减速器，承受大扭矩、用于重负载关节（如机器人肩/肘/髋）。双环传动是国内RV减速器代表。比谐波减速器便宜得多（PE 25 vs 519）。
报告用法：双环传动PE 25.5是减速器方向"最便宜"的标的——估值合理卡位核心的稀缺组合。
丝杠 — Lead Screw / Ball Screw 将旋转运动转化为直线运动的精密机械件，用于机器人线性关节。特斯拉Optimus大量采用丝杠关节，单机价值约3万元。拓普集团是特斯拉线性关节独家供应商。
报告用法：丝杠卡位★★★★，拓普集团PE 34.1/2026E PE 29.2——估值合理。
无框力矩电机 — Frameless Torque Motor 没有外壳框架的电机，体积小、扭矩大，直接嵌入机器人关节。汇川技术份额>50%，多环节卡位。是机器人电机的主流方案。
报告用法：汇川技术PE 37.1/PEG 1.48——电机方向"合理偏便宜"。
线控制动 — Brake-by-Wire 用电信号替代机械连接控制刹车，是自动驾驶L3+的必备硬件。伯特利是线控制动龙头，L3商业化直接受益。PE仅16.5倍，是物理AI产业链"最便宜"的标的。
报告用法：伯特利PE 16.5/PEG——下游智驾"最便宜"，L3商业化元年受益。
六维力传感器 — 6-Axis Force/Torque Sensor 同时测量三个方向的力和三个方向的力矩的传感器，让机器人具备"触觉"。国产化率仅30%，是机器人产业链替代空间最大的环节。
报告用法：六维力传感器国产化率仅30%——"替代空间最大"的零部件环节。
物料清单 BOM Bill of Materials 产品所有零部件的成本清单。BOM成本决定产品能否规模化量产——特斯拉Optimus BOM成本仍超6万美元，是量产元年"产业信号"黄灯的原因。
报告用法：Optimus BOM超6万美元——产业信号"🟢→🟡"，量产元年确认但成本未达标。
自动驾驶等级 L3 / L4 SAE Automation Level 3 / 4 SAE国际标准：L3=有条件自动驾驶（特定场景脱手，人需随时接管）；L4=高度自动驾驶（限定区域完全自主）。2026被定义为"L3商业化元年"。
报告用法：L3商业化+人形机器人量产，两条线同时进入"1到10爬坡期"。
